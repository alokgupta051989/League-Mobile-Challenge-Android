# League Android Code Challenge

This project showcases a progressive evolution of an Android application, starting from a simple MVVM structure to a professional-grade Multi-Module Clean Architecture.

## 🚀 Latest Version: 3.0 (Multi-Module Architecture)
The current stable version of the app features a fully modularized architecture, secure authentication using DataStore, and modern UI with Jetpack Compose.

**[Read Version 3.0 Documentation](README_v3.md)**

## 📜 Documentation History

- **[Version 3.0 (Current)](README_v3.md)**: Multi-Module, DataStore Auth, Advanced DI.
- **[Version 2.0](README_v2.md)**: Single Module Clean Architecture, Room Database, Paging 3 infrastructure.
- **[Version 1.0](README_v1.md)**: Basic MVVM with Retrofit.

## 🛠 Tech Stack (v3.0)
- **Architecture**: Multi-module Clean Architecture.
- **UI**: Jetpack Compose + Material 3.
- **DI**: Dagger Hilt.
- **Persistence**: Room + DataStore.
- **Networking**: Retrofit + OkHttp.
- **Concurrency**: Kotlin Coroutines & Flow.
